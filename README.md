#
### Deskripsi
Aplikasi yang dapat mendeteksi sebuah getaran
- Menggunakan Ms.Visual Express 2010
- Menggunakan Arduino IDE
- Mebutuhkan Arduino UNO, SW-420, dan Kabel Male-Female

### Fitur
- Connect
- Start/Stop
- Print
